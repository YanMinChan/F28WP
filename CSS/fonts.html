Font size

Font size can be modified by pixels(px), em, percentage and viewport width

Em and percentage is allowed to resize in browser menu (i.e. 1em is equal to current font size and will change with body %)

Viewport width font size will follow the size of the browser window (i.e. 1vw = 1% of viewport width)
